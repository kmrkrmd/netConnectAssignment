# netConnectAssignment
Download the repository,

unzip the file,

open command prompt,

set path to the unzipped directory,

Make sure python is installed,

install all libraries using requirements.txt,

open GDP Data.csv file here you can add,modify & Edit the price configurations,

run the code using "python main.py" command ,

open the browser and enter the url : http://127.0.0.1:5000/,

You can view the GDP in USD / GDP in INR By clicking on links.
